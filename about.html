About Ian

I'm Ian, and I build stuff. By trade, I'm a software engineer, but creating is a hobby to me as well as a career. I write code because I think software can have an incredible impact on people's lives, so I'm constantly coding up new ideas that I think could provide value.

By clicking around this site, you'll notice that I start lots of projects, and they're not all code related. In fact, a lot of my current projects are programming-adjacent. For example, I started a podcast called Greenpoint Dev where I and co-host Nikhil Dhawan discuss a variety of topics in software engineering. I've also been at work writing for a newsletter called LGTM that delivers weekly emails telling you tech news with a funny tone.

I love to share the things I create because I want my work to have a positive impact on others. What's the point of building something great that nobody uses? This is why I'm often energetic on social media. Starting conversions online and contributing to communities is a great way to get feedback on my projects and establish a platform where I can reach more people.

Feedback and meaningful conversations are critical when creating things that people will enjoy. If you send me an email, I will read it. If you want to get on a call and chat, just ask. You can even join my Discord server and start the conversation there.

I hope you enjoy your time on this site. You can learn more about the site by looking through the code in my portfolio GitHub repository. If you want to stay in touch, I write a monthly newsletter that gives updates on what I've been working on. I highly recommend subscribing for a quick and interesting read every month. In any event, thank you for the visit, and I hope you have a pleasant day.
